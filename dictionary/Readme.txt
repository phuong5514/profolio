biên dịch:

File main.Main.java có hàm main bắt đầu chương trình

chạy mã nguồn:
chạy file Main.java trong ide (cần thêm thư mục Resources chứa các file xml và 1 file chứa lịch sử history.xml có sẵn 1 thẻ <body>)
chạy file 22120287.jar trong folder Release: java -jar 22120287.jar (lưu ý thư mục Resources cần đặt tại nơi mở console để chạy file jar)
